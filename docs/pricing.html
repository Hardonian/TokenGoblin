Frequently asked questions

How does TokenGoblin track AI spend?

It attributes token usage per worker, per model, and per feature using deterministic rules against your actual usage data.

Can I start free and upgrade later?

Yes. Start on Starter, then upgrade to Pro or Enterprise via Stripe without switching tools.

What limits apply on each plan?

Starter is for small visibility with 10,000 events/mo. Pro adds forecasting, optimization, and 100,000 events/mo. Enterprise removes tenant and event limits.

Do you offer annual billing?

Annual plans are available. Use the enterprise contact path to set it up.

🎯 **What:** Extracted the overly long prompt fingerprint calculation from `BuildFingerprints` into a separate pure function `mapAccumulatorToFingerprint`. The `accumulator` struct was moved to package scope to support this.

✅ **Verification:** Ran tests with `go test ./internal/intelligence/...` and full `npm run test`, preserving identical behavior and passing type checks with `npm run typecheck` (`go vet` via `npm run lint`).
✨ **Result:** Improved modularity and readability of code without changing any existing functionality.
